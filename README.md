# 💅 Kathis Nagelstudio ✦

Ein VR-Spiel für die Meta Quest 3 — von PICZEL.

Du stehst in einem rosa Nagelstudio vor einer **riesigen Hand** und erfüllst Kundenwünsche:

- 🖌️ **8 Nagellack-Fläschchen** — greifen und die Riesen-Nägel anmalen
- ✨ **Glitzer-Streuer** — über dem Nagel schütteln
- 💅 **Feile** — macht die Nägel glänzend
- 🧽 **Schwamm** — wischt alles wieder ab
- ⭐ **Fertig-Knopf** — die Kundin bewertet dich mit Sternen, du bekommst **✦ Piczel**!

## So spielst du auf der Quest 3

1. Setz die Brille auf und öffne den **Browser** (das Weltkugel-Symbol).
2. Tippe diese Adresse ein: **https://bloxorn.github.io/KathisNagelstudio/**
3. Drücke unten auf **„ENTER VR"**.
4. **Trigger** (Zeigefinger-Knopf) = Werkzeug greifen und festhalten. Loslassen = zurücklegen.
5. Mit dem Pinsel über die Nägel streichen = malen. Fertig-Knopf drücken = Bewertung!

## Am Computer ausprobieren

Die gleiche Adresse im Browser öffnen. Mit der Maus auf ein Fläschchen klicken,
dann über einen Nagel ziehen. Ziehen ohne Werkzeug dreht die Kamera.

## Technik

- WebXR + [Three.js](https://threejs.org) — läuft ohne Installation direkt im Browser (Quest, Computer, überall)
- Kein Build nötig: `index.html` + `main.js`, fertig
- Piczel-Stand wird im Browser gespeichert
